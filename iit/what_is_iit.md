## 4. Exam Pattern (Very Important)

### JEE Main

* Computer-based test
* Multiple Choice + Numerical questions
* Tests **speed + accuracy**
* Used for:

  * NITs, IIITs, GFTIs
  * Qualification for JEE Advanced

### JEE Advanced

* **Very conceptual & tricky**
* Multiple correct answers
* Numerical value answers
* Match the following
* Tests **deep understanding**, not memory
